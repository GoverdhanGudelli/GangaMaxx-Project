Implementation of an independent, self-hosted logistics stack to eliminate 3rd-party API dependency and reduce scaling costs.

1. Live GPS Tracking (Traccar):
Integration with the Traccar open-source GPS tracking system for live fleet visibility, active geofencing, and support for hardware GPS devices in delivery trucks.

2. Dynamic Route Optimization (GraphHopper / OSRM):
Utilizing GraphHopper or OSRM over OpenStreetMap (OSM) data to enable dynamic route optimization and multi-stop sequencing based on specific delivery vehicle constraints (weight/height limits), solving the Traveling Salesman Problem with zero per-request API costs.

3. Interactive UI (Leaflet + OSM):
Using Leaflet.js combined with OpenStreetMap tiles to render custom, interactive delivery maps on the frontend dashboard without vendor lock-in.
